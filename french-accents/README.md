A simple package to conveniently type French with a non-French keyboard layout.
Text expansions match Vim's `:digraph` commands.

# Lowercase Accents
| Keyword | Replaced |
| --- | --- |
| `e'`| é |
| `` e` `` | è |
| `e^` | ê |
| `e:` | ë |
| `o^` | ô |
| `` a` `` | à |
| `a^` | â |
| `i^` | î |
| `i:` | ï |
| `` u` `` | ù |
| `u^` | û |
| `u:` | ü |
| `oe` | œ |
| `ae` | æ |
| `y:` | ÿ |
| `c,` | ç |

# Uppercase Accents
| Keyword | Replaced |
| --- | --- |
| `E'`| É |
| `` E` `` | È |
| `E^` | Ê |
| `E:` | Ë |
| `O^` | Ô |
| `` A` `` | À |
| `A^` | Â |
| `I^` | Î |
| `I:` | Ï |
| `` U` `` | Ù |
| `U^` | Û |
| `U:` | Ü |
| `OE` | Œ |
| `AE` | Æ |
| `Y:` | Ÿ |
| `C,` | Ç |
